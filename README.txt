Gully Cricket Pro - Pure HTML Project

How to run:
1. Extract this zip.
2. Open index.html in Chrome/Edge.
3. No npm, no React, no build needed.

Features:
- Team and player setup
- Custom overs
- Last-man batting rule
- Retire-at-runs rule
- Striker/non-striker/bowler control
- Runs, wide, no-ball, bye, leg-bye, wicket
- Undo last ball
- End innings manually
- Scorecard, timeline, match summary
- Auto-save using browser localStorage
