# Gully Cricket Ultra

A premium single-file HTML cricket scoring app built for gully cricket matches with an IPL-style live scoring experience. The app includes team setup, captain selection, toss flow, striker and non-striker setup, ball-by-ball scoring, wicket handling, innings change, target chase, scorecard, records, and final match result.

## Features

* Single HTML file project
* Fully responsive mobile-first layout
* IPL-style scoreboard UI
* Team name and player lineup setup
* Captain selection
* Automatic toss system
* Toss call and toss result display
* Batting or bowling decision after toss
* Striker and non-striker selection
* Bowler selection
* Ball-by-ball scoring
* Auto strike rotation on singles and odd runs
* Auto strike change after over completion
* Next bowler popup after every over
* Wicket popup with dismissal details
* Next batter selection after wicket
* Wide, no-ball, bye, leg-bye support
* This over and last over display
* Innings end popup with target
* Match end result screen
* Scorecard with batting and bowling details
* Match records and summary
* LocalStorage support for saving match progress

## Tech Stack

* HTML
* CSS
* JavaScript
* LocalStorage

## How to Run

Open `index.html` directly in any browser.

No installation required.

## GitHub Pages Deployment

1. Push this project to GitHub.
2. Open the GitHub repository.
3. Go to **Settings**.
4. Open **Pages**.
5. Select **Deploy from a branch**.
6. Choose branch: `main`.
7. Choose folder: `/root`.
8. Click **Save**.

Your website will be live after a few minutes.

## Project Goal

This project is designed to make local gully cricket matches feel like a professional live cricket broadcast. It helps players manage match setup, scoring, wickets, innings, and results in a clean and easy-to-use interface.
